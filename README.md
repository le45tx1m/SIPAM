一、前言

SIPAM主要用于解决工作中需要对多个网段的IP地址进行分配的工作  

软件基于C#开发的PC客户端和mariaDB  

目前市面上也有开源的ipam系统，但是功能有点复杂，不适合我这种需求简单的用户

所以我自己就写了这么一坨东西，目前处于测试版，功能尚不完善，使用需谨慎！使用需谨慎！使用需谨慎！

二、软件如何使用

第一步：准备一个MariaDB（MySQL）数据库

第二步：将数据库信息填入软件

第三步：创建基础数据即可
